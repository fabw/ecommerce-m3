# Ecommerce CLI - Módulo 3

Aplicación de consola desarrollada en Python para simular el funcionamiento básico de un ecommerce.

## Funcionalidades

- Ver catálogo de productos
- Buscar productos por nombre o categoría
- Agregar productos al carrito
- Ver carrito con subtotales y total a pagar
- Vaciar carrito
- Salir del programa
